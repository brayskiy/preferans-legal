# preferans-legal

Public mirror of legal documents for the **Preferans** Android app
(`com.rayskiy.preferans`).

This repository exists solely to host the privacy policy at a stable public URL
via GitHub Pages — the Preferans source code lives in a separate repository
(<https://github.com/brayskiy/preferans>).

## Live URLs

- Landing: <https://brayskiy.github.io/preferans-legal/>
- Privacy Policy: <https://brayskiy.github.io/preferans-legal/privacy.html>

## Editing

The canonical text lives as Markdown in the Preferans repository's
`docs/PRIVACY.md`. The HTML file here is rendered from that source whenever
the policy changes.
